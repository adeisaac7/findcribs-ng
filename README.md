# FindCribs.ng
A Nigerian real estate platform connecting buyers, renters, and property agents.

🔗 [Live Site](https://findcribs.ng)

## About
FindCribs helps users find vacant apartments, houses, and properties across Nigeria.  It provides users with a seamless experience for browsing and listing property listings. Our goal is to simplify the process of finding and managing properties.

## Tech Stack
- Next.js 16
- TypeScript
- TailwindCSS
- REST APIs
  

## Role
Redesign, frontend development, authentication, admin panel, SEO, and backend integrations.
